This demo is about PBR and 3D SoftBody physics based on Visual Studio 2019 using OpenGL and GLSL shader codes
Basically, there are 6 scenes from scene0 to scene5 and you can change by UI
In Scene 1,2,3,4,5 you can move the camera by arrow keys and change the angle by W,A,S,D keys
But in Scene 0, the camera is stick to the main object
Moreover, you can see FPS on every scene and each scene has a little different components to control by UI
